# HealthyFood - Сервис доставки здорового питания

## Описание

**HealthyFood** — лендинг сервиса доставки здорового питания на фреймворке **Tailwind CSS**.

Проект создан в рамках практических занятий №7 и №8.

### Тема
Сервис доставки готовых рационов здорового питания

### CSS-фреймворк
**Tailwind CSS v3** - utility-first CSS-фреймворк через CDN

#### Почему Tailwind?
- ✅ Utility-first подход
- ✅ Быстрая разработка
- ✅ Полный контроль над дизайном
- ✅ Адаптивность из коробки
- ✅ Кастомизация через конфиг
- ✅ JIT-компиляция
